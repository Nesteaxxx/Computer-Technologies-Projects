## Описание

Проект включает несколько модулей для различных задач компьютерного зрения:
- Обработка цвета и трансформация изображений
- Кластеризация и сегментация изображений
- Обнаружение контуров и геометрических фигур
- Классификация объектов на основе признаков
- Анализ биометрических данных (зрачки и блики света)

## Модули проекта

### 1. 🍎 **apples/** - Трансформация цвета
**Файл:** `redappletogreen.py`

Модуль для детектирования красных яблок и преобразования их цвета в зелёный.

**Функциональность:**
- Детектирование красных объектов в HSV цветовом пространстве
- Фильтрация по цвету и яркости
- Морфологические операции (дилатация)
- Преобразование красного цвета в зелёный с сохранением текстуры

**Основные операции:**
- Конвертация BGR → HSV
- Создание масок для красного и зелёного цветов
- Анализ контуров
- Реколорирование объектов

---

### 2. 🔍 **claster/** - Кластеризация изображений
**Файл:** `claster.py`

Реализация k-means кластеризации для сегментации изображений в полутонах и цвете.

**Функциональность:**
- Кластеризация изображений в полутонах (grayscale)
- Кластеризация цветных изображений (RGB)
- Расчёт процентного соотношения кластеров
- Визуализация результатов рядом с оригиналом

**Функции:**
- `kmeans_clustering_grayscale()` - кластеризация чёрно-белых изображений
- `kmeans_clustering_color()` - кластеризация цветных изображений

---

### 3. 💰 **coins/** - Детектирование монет

#### 3.1 `classification.py` - HOG дескриптор
Реализация Histogram of Oriented Gradients (HOG) для извлечения признаков из изображений.

**Функциональность:**
- Вычисление градиентов (Sobel)
- Построение гистограмм направленных градиентов
- Нормализация дескрипторов
- Использование для классификации объектов

**Параметры:**
- `win_size` - размер окна (128×128)
- `cell_size` - размер ячейки (8×8)
- `nbins` - количество бинов гистограммы (9)

#### 3.2 `contour&square.py` - Контуры и площади
Обнаружение контуров монет и расчёт их площадей.

**Функциональность:**
- Преобразование изображения в чёрно-белое (бинаризация)
- Размытие Гаусса для сглаживания
- Морфологические операции (дилатация)
- Применение оператора Собеля для выделения краёв
- Поиск и анализ контуров
- Расчёт и отображение площадей объектов

---

### 4. 👁️ **pupils&glare/** - Анализ зрачков и бликов
**Файл:** `findpupils&glare.py`

Детектирование зрачков и бликов света (зуб) в изображениях глаз.

**Функциональность:**
- Распознавание зрачков в изображениях глаз
- Детектирование бликов света (glare)
- Использование каскадного классификатора Haar для поиска глаз
- Анализ циркулярности контуров
- Сравнение трёх версий изображения для выделения артефактов

**Компоненты:**
- `get_circularity()` - вычисление циркулярности контура
- `process_single_eye()` - обработка одного изображения глаза
- Использование Cascade Classifier для начальной локализации
